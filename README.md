# Task 1: Iris Flower Classification

## Project Overview

The **Iris Flower Classification** project is a classic machine learning classification problem. The goal of this project is to build a machine learning model that can classify iris flowers into three species based on their measurements.

The three species included in the dataset are:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

The model learns patterns from the flower measurements and predicts the species of a new flower.

## Dataset

Dataset Source: Kaggle

Features included in the dataset:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (Target Variable)

## Workflow

1. Import required libraries
2. Load and explore the dataset
3. Perform data preprocessing
4. Split dataset into training and testing sets
5. Train a machine learning classification model
6. Predict the species of iris flowers
7. Evaluate the model accuracy

## Result

The trained model successfully classifies iris flower species with high accuracy using machine learning techniques.

## Dataset

Dataset Source: Kaggle

Features included in the dataset:

* Region
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area
* Longitude
* Latitude

## Workflow

1. Import required libraries
2. Load the dataset
3. Perform data cleaning
4. Handle missing values
5. Analyze unemployment statistics
6. Visualize unemployment trends using graphs
7. Identify patterns and insights

# Task 3: Car Price Prediction with Machine Learning

## Project Overview

The objective of this project is to build a machine learning model that can **predict the price of a used car** based on various factors such as brand, fuel type, transmission type, mileage, and other attributes.

Car price prediction is an important application of machine learning in the automobile industry.

## Dataset

Dataset Source: Kaggle

Features in the dataset include:

* Car Brand
* Model
* Year
* Fuel Type
* Transmission
* Mileage
* Engine Specifications
* Car Price (Target Variable)

## Workflow

1. Import required libraries
2. Load the dataset
3. Explore dataset structure
4. Handle missing values
5. Convert categorical data into numerical form
6. Split dataset into training and testing sets
7. Train a regression model
8. Predict car prices
9. Evaluate model performance

## Model Used

* Linear Regression

## Evaluation Metrics

* Mean Squared Error (MSE)
* R² Score

## Result

The machine learning model successfully predicts car prices based on the provided features, demonstrating the application of regression techniques in real-world scenarios.
